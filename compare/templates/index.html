How it works:

You put your shopping list in at the top, this will tell you where the cheapest place is to shop.

Webscrapers use Selenium and BeautifulSoup4 to update PostgreSQL database hosted on bit.io. Django site was hosted on AWS EC2 instance. UI with Node.js.

This a Proof-of-Concept MVP. More items and functionality are being added all the time.
